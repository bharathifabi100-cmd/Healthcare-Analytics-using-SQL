# 🏥 Healthcare Analytics using SQL

# 📌 Project Overview

This project demonstrates **Healthcare Data Analytics using SQL** by analyzing patient records, doctor interactions, appointments, diagnoses, and medications.

The project focuses on writing advanced SQL queries to solve real-world healthcare business problems and generate actionable insights.

---

# 🎯 Business Problem

Healthcare organizations generate massive amounts of patient data every day.

The objective of this project is to use SQL to answer business questions such as:

- Which doctors treat the most patients?
- Which patients require continuous care?
- What are the most common diagnoses?
- Which days are busiest for appointments?
- How can hospitals optimize workload and patient management?

---

### 👨‍👩‍👧 Patient Analysis
